Slackware

Sada postoji mogucnost da vam nije bootalo sistem

Sve zavisi na koju ste masinu installirali i koju verziju

Znam da je frustrirajuce, meni se to desilo 100 puta

Terminal

Poslije urednog bootanja utipkajte pod user:root

Utipkajte:password koji ste sami izabrali prilikom zavrsetka instalacije.

adduser

Prvo cemo da kreiramo novog korisnika

Utipkajte: adduser

Sledece sto cete da vidite je:

Login name for new user []:

Ukucajte vas ime ili sta zelite(Enter)

Sledece korake samo Enter-Enter do password-a

ili ipak mozete na neka pitanja da odgovorite po zelji vam

Kada dodje New password utipkajte password

Zatim ce ponovo da trazi da utipkate isto

Sledece tipkajte: exit

Sada ste uradili log-out kao root user

Sledeci korak je da se ulogujemo kao novi user

Pa unesite novo korisnicko ime i novi password

Sada kada ste ulogovani pod novim korisnikom

Vas sistem je siguran da ga necete pokvariti :)

A i poslije kad steknete vise iskustva nikada necete raditi pod user root account

Idemo dalje-Ne zelim ni da pomisljate na GUI

Neke od komandi

Sve ove komande od danas cete koristiti stalno

Kucajte pwd

Printa trenutni put do faseg foldera u kojem se nalazite

Nadam se da vam je jasno gdje se nalazite poslije isprintanog

Utipkajte sada cd /

Zatim utipkajte ls

Da li vidite sada:

bin boot dev etc home lib lost+found media mnt opt proc root run sbin srv sys tmp usr var

cd / gdje god da se nalazimo ta komanda nas vraca u root direktorij

ls - izlistava sve sto se nalazi unutar foldera u kojem se nalazimo

Probajte sada ls -l, ls -t, ls -a, ls -b, ls -F, ls -al

Kada tipkate ls -al tada cete da vidite pored svakog foldera ili fajla nesto kao-rw-r--r--

To vam govori u kojem modu je neki folder ili fajl, to vam prepustam da samo naucite sta je chmod komanda

chmod cete koristit kada se susretnete kao root user u neki fajl i prikaze Permisson Denied

Slededa komanda je cd /home

Prelazite u home folder, znacenje cd je change direktorij

Sada kucajte ls i pogledajte sta sve ima u /home

Sada samo predijte u jedan od foldera koji se nalaze unutra

Ovako: cd /ime_tog_direktorija

Kada ste presli u taj folder opet tipkajte ls

Sada ukucajte ../

Ponovo ls

Skontali ste da komanda ../

Sta??? Vraca nas jedan folder unazad

Sto se mene tice naucili ste kretanje iz foldera u folder

Meni dovoljno da krenemo na Step-3 vim

Vi cete sami da nastavite ovde Sve o vim-u
